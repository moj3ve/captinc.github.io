DarkModeToggle

Enable/disable specific tweaks when toggling dark mode
Compatible with iOS 11.0+
Configure options in Settings after installing

Source code
Reddit post

Features:

  • Customize exactly which tweaks should be on/off in dark/light mode
  • Works with the stock iOS 13 dark mode module
  • Provides a custom module for use with iOS 12 and older
  • After toggling dark mode, kill your apps or respring to enable/disable your tweaks
  • Optionally run a shell script as root afterwards (useful for changing your wallpaper with dark mode)
  • Compatible with WallpaperChanger and YouTubeDarkModeSync

Main uses:

  • Toggle app-specific dark modes with iOS 13's dark mode
  • For iOS 12, you can have one toggle that turns all dark mode tweaks on/off
  • When combined with WallpaperChanger, you can have custom appearance-aware wallpapers
