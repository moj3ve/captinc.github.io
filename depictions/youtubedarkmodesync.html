YouTubeDarkModeSync

Toggle YouTube's built-in dark mode with iOS 13's dark mode
Compatible with iOS 13.0+

Reddit post
Source code

Features

  • Toggles YouTube's built-in dark mode whenever you toggle iOS 13's dark mode
  • Compatible with CercubeDarkMode
  • Useful with DarkModeToggle (FYI, YouTubeDarkModeSync adapts with iOS 13 already, so there's no need to explicitly disable it in DarkModeToggle)
