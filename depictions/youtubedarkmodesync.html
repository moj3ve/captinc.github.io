YouTubeDarkModeSync

Toggle YouTube's built-in dark mode with iOS 13's dark mode, DarkModeToggle, and Noctis
Compatible with iOS 11.0+

Reddit post
Source code

Features

  • Toggle YouTube's built-in dark mode with iOS 13's dark mode, DarkModeToggle, or Noctis12 (in that order)
  • Respects iOS 13's dark mode if you're on iOS 13. If not, it respects DarkModeToggle. If DarkModeToggle is not installed, it respects Noctis12
  • Compatible with CercubeDarkMode
  • Note - as of YouTube v15.03.1, stock YouTube toggles with iOS 13's dark mode. However, this tweak is still fully functional if you want support for DarkModeToggle/Noctis/iOS 12
